# Reinforcements

`Reinforcements` is a node js package written in `Typescript` to give a massive support for variant data like `Strings`, `Arrays`, `Objects` ... and so on.


# Installation

`yarn add @mongez/reinforcements`

or using `npm`

`npm i @mongez/reinforcements`
